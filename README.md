# Installation:
## First, clone this repository.
```bash
git@github.com:fandredev/angular-18-features.git
```

## Install dependencies.
```bash
cd angular-18-features && npm install
```
 
## Run development server
```bash
npm run start
```
- or using ```launch.json```  file (if you use vscode)

## Run tests
```bash
npm run test
```
